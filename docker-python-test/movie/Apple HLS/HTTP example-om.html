Anvisningar för postning av HTTP example.m3u8 på din webbplats

Du har exporterat HTTP example.m3u8 med Compressor, Final Cut Pro eller Motion. Följande information hjälper dig att posta filmen på din webbplats i samma mapp som den här HTML-filen med instruktioner.

Det finns även en HTTP-strömspellista (.m3u8) i mappen som väljer bäst version för dina besökares olika enheter och nuvarande anslutningshastigheter. Genom att inkludera denna spellista på din HTML-sida ser du till att videon på webbplatsen fungerar suveränt för alla.

Compressor har placerat de angivna versionerna av filmen enligt dina måls specifikation.

Använd följande HTML-kod till att bädda in filmen på webbsidan.

Viktiga anmärkningar:

  • Om du vill dela med dig av medier via internet måste dina mediefiler (filmer, vinjettbilder) finnas på en server som allmänheten har tillgång till.
  • URL-parametrarna måste ändras om dina mediefiler ligger på en annan server än din webbsida.
  • Filmens URL-parametrar måste alltid peka på strömspellistan.
  • Sökvägarna i spellistan (.m3u8) bör vara relativa gentemot spellistan och strömsegmenten.
